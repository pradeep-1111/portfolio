🤖 ML Model · Live App

ML Vulnerability Severity Classifier

Hierarchical 3-stage scikit-learn classifier trained on 55,712 NVD & CVE records. Predicts CVE severity (Critical / High / Medium / Low) with 71.2% accuracy & 0.716 weighted F1. Combines TF-IDF (60k features) over CVE descriptions with one-hot CVSS metrics, deployed as a Streamlit app with RAG similarity search, batch prediction, and CSV export.
